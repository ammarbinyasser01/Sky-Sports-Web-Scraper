# Sky Sports Web Scraper

A simple Python web scraper that fetches and displays the latest headlines from the Sky Sports website using **Requests** and **BeautifulSoup**.

## Features

* Fetches the Sky Sports homepage
* Parses HTML using BeautifulSoup
* Extracts the latest news headlines
* Displays the headlines in the terminal

## Requirements

* Python 3
* requests
* beautifulsoup4

Install the required libraries:

```bash
pip install -r requirements.txt
```

## Run the Project

```bash
python news_scraper.py
```
